# gulp-switch

Stream condition for switch/case, gulp plugin that is shim to [stream-switch](https://github.com/popomore/stream-switch)

---

## Install

```
$ npm install gulp-switch -g
```

## Usage

```
var gulp-switch = require('gulp-switch');
```
